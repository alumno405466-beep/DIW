## [TAREA 1.3](https://github.com/alumno405466-beep/DIW/tree/main/1.3)
El repositorio GITHUB contiene:
- Fichero SCSS
- Fichero CSS
- Fichero HTML
- Carpeta sassdoc
Vuestro archivo SCSS, debe tener, como mínimo:
- Comentarios SassDoc, sobre las variables, estructuras de control, mixins y funciones.

- Definición de variables al inicio del documento (Todos los valores usados, deberán estar indicados como variables).
- Uso de un mapa o una lista.
- Estructuras de control IF/ELSE
- Estructuras de control WHILE
- Estructuras de control FOR
- Estructuras de control EACH
- Uso de una función definida por el usuario.
- Uso de las directivas @error / @warn / @debug
- Uso de un mixin
